Mean Features correlations

'radius_mean','perimeter_mean','area_mean','concavity_mean','concave_points_mean' are correlated with diagnosis so we could use them to create the model.

Even 'radius_mean', 'perimeter_mean', 'area_mean' are highly correlated with each other, we will still use all of them.

Model Classification

SE Features correlations

Only 'radius_se', and 'perimeter_se' are just slightly correlated with diagnosis so we will not use the se features to create the model.

'radius_worst', 'perimeter_worst', 'area_worst', 'concavity_worst', 'concave-points_worst' are correlated with diagnosis so we will use them to create the model. Even 'radius_worst', 'perimeter_worst', 'area_worst' are highly correlated with each other we will use all of them to create the model.

Worst Features correlations

'radius_worst', 'perimeter_worst', 'area_worst', 'concavity_worst', 'concave-points_worst' are correlated with diagnosis so we will use them to create the model.

'radius_worst', 'perimeter_worst', 'area_worst' are highly correlated with each other but we will still use all of them to create the model.
